# Heyproxy Website

Official website for Heyproxy - Expert Web3 Development, Smart Contracts & Blockchain Solutions.

## Features

- **Modern Design**: Sleek, dark-themed UI with cyan accents and smooth animations
- **SEO Optimized**: Comprehensive meta tags, Open Graph, structured data for Web3/blockchain keywords
- **Responsive**: Fully responsive design that works on all devices
- **Performance**: Optimized code with minimal dependencies
- **Accessibility**: Semantic HTML and ARIA labels

## Tech Stack

- **HTML5**: Semantic markup
- **CSS3**: Custom properties, animations, and modern layouts
- **TypeScript**: Type-safe JavaScript with compilation to ES6
- **No Framework**: Vanilla JS for maximum performance

## Installation

```bash
# Install dependencies
npm install

# Build TypeScript
npm run build
```

## Deployment

The website is a static site and can be deployed to any static hosting service:

- GitHub Pages
- Netlify
- Vercel
- AWS S3 + CloudFront
- Any web server

Simply upload all files to your hosting provider.

## Files Structure

```
├── index.html          # Main HTML file
├── styles.css          # All styles
├── src/
│   └── main.ts        # TypeScript source
├── dist/
│   └── main.js        # Compiled JavaScript
├── sitemap.xml        # SEO sitemap
├── robots.txt         # Search engine directives
└── *.svg              # Logo files
```

## SEO

The website includes:
- Meta tags optimized for Web3, blockchain, and smart contract keywords
- Open Graph tags for social media sharing
- Twitter Card tags
- Structured data (JSON-LD) for organization information
- XML sitemap for search engines
- robots.txt for crawler directives
